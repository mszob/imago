Dvorak International is a keyboard layout for macOS which makes it easier for Dvorak users to type certain diacritics. It does this by imitating the sams system of dead-keys which many QWERTY users know thanks to the U.S. International - PC layout.

The layout is also QWERTY-command.

|  Download Dvorak International  |
|                                 |
`---------------------------------’

The .zip archive contains a .bundle file which needs to be placed in /Library/Keyboard Layouts directory.
