Obsidian

The following keyboard layouts offer practical functionality to Dvorak typists on macOS, as well as users of small keyboards.

Each individual layout can be downloaded as a .zip archive which contains a .bundle file. This file needs to be placed in the /Library/Keyboard Layouts directory in macOS. 

+---------------------------------+
|                                 |
|      Download all layouts       |
|                                 |
+---------------------------------+

Alternatively, all layouts can be downloaded simultaneously above.

Dvorak International

Dvorak International is a keyboard layout for macOS which makes it easier for Dvorak users to type the following diacritics. It does this by imitating the sams system of dead-keys which many QWERTY users know thanks to the U.S. International - PC layout. 

+---------------------------------+
|                                 |
|  Download Dvorak International  |
|                                 |
+---------------------------------+

The layout is also QWERTY-command.

30% International

This layout is based on the U.S. International layout but is modified for small keyboards (e.g. 30% or 40%). Is uses the semi-colon as a dead key for the circumflex accent, rather than the caret symbol, as the latter is not as accessible with small form-factors.

+---------------------------------+
|                                 |
|   Download 30% International    |
|                                 |
+---------------------------------+

This layout is based on QWERTY since it is meant to be used with an external keyboard; any any alternate layouts will are used directly on the external keyboard's controller.

Dvorak to QWERTY

This layout allows an external Dvorak keyboard to be used as a QWERTY keyboard even if a QWERTY layout is not included in the external keyboard's firmware.

+---------------------------------+
|                                 |
|    Download Dvorak to QWERTY    |
|                                 |
+---------------------------------+

The layout does this by simply mapping a QWERTY key's position to the QWERTY letter which occupies the position of that same key in the Dvorak layout.
