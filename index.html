The following keyboard layouts offer practical functionality to Dvorak typists and users of small keyboards on macOS.

Installation

Each individual layout can be downloaded as a .zip archive which contains a .bundle file. This file needs to be placed in the /Library/Keyboard Layouts directory, then installed in System Settings > Keyboard > Keyboard layouts. Click on the + button to add a new input source, then locate custom layouts in the Others category.

Alternatively, all layouts can be downloaded simultaneously above.

Dvorak International

Dvorak International makes it easier for Dvorak users to type a number of diacritics and accents used in French, Spanish, and other languages. It does this by imitating the same system of dead-keys which many QWERTY users know thanks to the U.S. International - PC layout. 

+---------------------------------+
|                                 |
|  Download Dvorak International  |
|                                 |
+---------------------------------+

The layout also includes QWERTY commands.

30% International

30% International is a diacritics-focused layout based on U.S. International with small keyboards in mind (e.g. 30% or 40%). Is uses the semi-colon as a dead key for the circumflex accent, rather than the caret symbol, the latter being not as accessible with small form-factors. 

+---------------------------------+
|                                 |
|   Download 30% International    |
|                                 |
+---------------------------------+

Since it is meant to be used with external keyboards, third layout is QWERTY; any alternate layouts are implemented directly on the external keyboard's controller.

Dvorak to QWERTY

Dvorak to QWERTY allows an external Dvorak keyboard to be used as a QWERTY keyboard even if the QWERTY layout is not included in its firmware.

+---------------------------------+
|                                 |
|    Download Dvorak to QWERTY    |
|                                 |
+---------------------------------+

The layout achieves this conversion by mapping a QWERTY key's position to the QWERTY letter which occupies the position of that same key in the Dvorak layout.

I use this custom English script to explore aspects of writing which I find interesting. I try to make it 1] a bit more phonetic than standard English spelling, 2] fun to write in cursive!

It is a work in progress.

Consonant sounds

The table below summarizes how the script approaches organizing consonants.

1. Sounds are organized into three families based on the general area where they are pronounced in the mouth (Y axis).

2. Within the families, sounds are organized in series based on three factors of similarity (Z axis):

  • Unvoiced and voiced variation on the same sound (e.g. t/d)
  • Similar pronunciation across at least two components (e.g. both t/d pair and θ/đ pair are pronounced with the tip of the tongue at the front of the mouth)
  • Isolated and compound occurrences of the same sound (tʃ/dʒ pair contains t/d sounds)

Writing consonants

The general character shape of written consonant sounds indicates location in the mouth; each individual charactes indicates a series of similar sounds; and diacritics distinguish each sound in the series.
